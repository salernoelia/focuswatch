# Releases

## [1.2.1] - 01.12.2025

#### Changes
- Micro-apps configuration options from the Wizard
- Rename "XP" to "Points"
- Level Up "Current Level * 100" amount of Focus Points
- The Focus Meter wheel is now not red->orange->green anymore but instead orange->green<-orange
- Improved Phone and Watch connection stability and many background synchronisation capabilities
- Added missing translations for DE

#### What to test

- Validate which features create real value for daily use, focusing on writing detection, checklists, mood capture, and focus tools
- Check usability and clarity of basic actions, including starting tasks, following checklists, and reacting to reminders
- Observe engagement patterns, especially whether children pick up the watch independently and use features without prompts
- Identify friction points, for example confusion, false detections, weak motivation, or ineffective nudges
